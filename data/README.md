# Sample Data

Place your PDF document here for testing the RAG frameworks.

## Recommended Test Documents

- Technical papers or research articles (10-20 pages)
- Company reports or whitepapers
- Product documentation
- Any substantive PDF with searchable content

## Usage

1. Add your PDF file to this directory
2. Rename it to `sample.pdf` or update the file paths in the example scripts
3. Run the examples or comparison script

## Example Sources

- [arXiv](https://arxiv.org/) - Open access research papers
- [Papers With Code](https://paperswithcode.com/) - ML research papers
- Company investor relations pages - Annual reports
- Government agency publications

## Important Notes

- Ensure the PDF has selectable text (not scanned images)
- Larger documents will take longer to process
- The same document is used across all three frameworks for fair comparison
